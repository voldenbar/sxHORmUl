# 项目简介

基于深度学习的虾病害检测系统，运用YOLOv10算法与自定义YOLO数据集，旨在实现对虾养殖过程中病害的快速识别与预警。系统包含UI界面，用户可直观操作，实现图片与实时视频流中虾病害的检测。采用Python开发，项目源码开放，便于技术交流与模型优化。技术特点包括高检测精度、低延迟以及易于部署。该系统作为一项技术探索，助力养殖业的智能化发展。


## 核心功能
- 基于YOLOv10算法与自定义YOLO数据集实现虾病害检测
- 支持图片与实时视频流中的病害识别
- 提供直观操作的UI界面
- 开放Python项目源码，便于技术交流与模型优化
- 实现病害的快速识别与预警功能

## 适用场景
- 虾养殖过程中的病害监测
- 养殖业的智能化管理
- 技术研究与模型优化
- 养殖环境中的实时监控

## 技术特点
- 高检测精度
- 低延迟处理
- 易于部署
- 基于深度学习的智能识别技术
- 支持自定义数据集训练
